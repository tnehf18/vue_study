4. 리스트 렌더링

반복문 v-for

● 변수 in 정수

  v-for="i in 11"

  일반적인 반복문, 인덱싱에 사용될 변수를 i 외에 사용해도 되는지, 또는 데이터 바인딩을 통해 vue 인스턴스 안에 만들어진 변수를 사용가능한지 등이 궁금함.
  (교재에서 구구단이 나오는걸 보니, 이 교재가 전문가보다는 10대 중반의 청소년을 대상으로 한 것일 수도 있겠다는 생각이 든다...)
  > 테스트 해봤으나 정수형으로 단순 반복할 때는 상관없이 독립적으로 수행되는 듯함.

● 요소 in 배열
  
  data: {
    stories: [
        "차를 부숴먹었어요!",
        "어제 누가 내 가방을 훔쳐갔어!",
        "누가 내 초콜릿을 먹었어...",
    ]
  }

  ...

  v-for="(value, index) in stories"
  
  v-for 내부에 바인딩 된 데이터 중 배열을 가져왔을 때의 문법이 python과 비슷하다는 생각이 든다.
  
  각 요소가 객체일 경우에는 객체 그 자체가 변수에 담기는데, 거기서 꺼낼 수 있는 건 편리해 보인다.
  

● 프로퍼티 in 객체

  data: {
      story: {
          plot: "차를 부숴먹었어요!",
          writer: "Alex",
          upvotes: 47
      }
  }

  ...

  v-for="(val, key, index) in story"

  객체로 선언된 데이터도 가져올 수 있는데,
  반복문의 인자의 순서를 바꿔도 인식할 수 있나 테스트해봤는데, 그렇게는 안되므로 인자 순서가 중요한 것 같다.

  ※ 인자의 순서에 주의.
